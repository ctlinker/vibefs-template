vibefs-template v0.0.0

Drop in.
Serve it.

Minimal Chi-based static app template. Drop your frontend in ./public and serve it under a configurable path.

$ go run main.go // start server
$ APP_PATH=/app go run main.go // custom path
project layout
.
├── go.mod
├── main.go // config-aware server
├── public/
│ └── index.html // ← you are here
└── src/config/
├── conf.go
└── load.go
Replace this file with your app — chi handles the rest.
